#                项目名称：商城

##### 本项目是一个前后端分离项目 前台有LayUI做界面Vue数据渲染 后台SpringBoot+Swagger2 做api接口  JWT token做权限验证

## 技术栈

- SpingBoot

- Mybatis

- Mysql

- Redis

- JWT

- SMS

- Cookie购物车

- 支付宝沙箱

- Vue

- LayUI

- ES6

- LNMP

  

  

## API接口 ：http://mall.lqcnb.cn/swagger-ui.html

## 项目声明：本项目为学年设计 不能用作商业用途
